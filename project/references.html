Development Forge

The digital archives, tools, and platforms utilized to forge the code, architecture, and design of the Bloodline project.

Documentation & Learning

  • MDN Web Docs

    The primary blueprint for semantics and CSS layout architecture. Used heavily for foundational cross-browser compatibility guidelines.

  • W3Schools

    The quick-reference scroll for rapid syntax validation, semantic HTML templates, and foundational element structures.

  • Codecademy

    The training grounds where the core fundamentals of structured styling and interactive logic were first mastered.

Design, Color & Experimentation

  • Coolors

    The canvas utilized to extract, lock, and balance the dark crimson, muted gold, and charcoal palette of the Bloodline realm.

  • CodePen

    A sandbox environment used for isolating, testing, and perfecting advanced CSS transitions and layout experiments before deployment.

  • Google Fonts

    Provider of the typography, combining the sharp elegance of Cinzel with the clean legibility of Montserrat.
